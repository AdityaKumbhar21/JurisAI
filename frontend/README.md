###  **Getting Started**  
1. Clone the repository:  
```bash
git clone <your-repo-url>
cd jurisAI-frontend
```
2. Install dependencies:  
```bash
npm install
```
3. Run the app:  
```bash
npm run dev
```

---

### **Core Features**  
- Upload documents (PDF or Image formats).  
- Poll for summarization status in real-time.  
- View summaries of uploaded documents.  
- View upload history.
